# IS111-Intro-to-Programming

SMU Module - AY22/23

> TA documents will be separated and private.

### Learnings

-   What it takes to write code
-   Logical thinking
-   Code with Python

### About

Taken in AY22/23 Semester 1, IS111 is designed to be an entry-level programming module for Information System Freshmen.

-   It is taught in Python, but the knowledge can be applied into other programming languages.

-   Question difficulty is distinguished by Start-Level, 1-3 Stars

-   The objective of this modules is to expose programming to newer programmers, and give experienced students a taste of real-world coding problems
